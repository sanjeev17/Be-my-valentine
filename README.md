# Be My Valentine

This project was designed and built by me to meet all requirements for a Valentine themed interactive website.

## Overview

- A playful landing page with floating hearts, smooth button motion, and a cheeky "No" button.
- A sweet "Yes" page for the final reveal.
- Mobile friendly layout and touch support.

## Requirements covered

- Single page entry with an interactive prompt.
- Smooth UI transitions and animated elements.
- Responsive design for desktop and mobile.
- Clean React + TypeScript structure with Vite.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Getting started

```sh
# Install dependencies
npm install

# Run the dev server
npm run dev
```

## Scripts

- `npm run dev` - start local dev server
- `npm run build` - production build
- `npm run preview` - preview production build
- `npm run test` - run tests
- `npm run lint` - run ESLint

## Deployment

Build the app and deploy the `dist/` folder.

```sh
npm run build
```
